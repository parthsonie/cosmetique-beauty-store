# Cosmetique

This is a restructured version of the original college HTML and CSS project.

## Pages

- `index.html` - Home page
- `service.html` - Beauty services page
- `shop.html` - Shop page with a small React product grid enhancement
- `login.html` - Login page
- `signup.html` - Sign up page

## Tech Used

- HTML
- CSS
- JavaScript
- A small amount of React on the shop product grid

Open `index.html` in a browser to view the site.
